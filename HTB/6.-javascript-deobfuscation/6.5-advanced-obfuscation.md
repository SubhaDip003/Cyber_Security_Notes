# 6.5 Advanced Obfuscation

Basic obfuscation hides code partially, but some readable text or strings may still remain. Advanced obfuscation is used to completely hide the original logic, variable names, and strings so the code becomes extremely difficult for humans to understand while still running normally in the browser.

## Obfuscator <a href="#obfuscator" id="obfuscator"></a>

One popular tool for this is [**obfuscator.io**](https://obfuscator.io/). It converts normal JavaScript into a complex and encoded format. Before we click `obfuscate`, we will select `Encoding` to `Base64`, as seen below:

There are many other JavaScript obfuscators, like [JJ Encode](https://utf-8.jp/public/jjencode.html) or [AA Encode](https://utf-8.jp/public/aaencode.html). However, such obfuscators usually make code execution/compilation very slow, so it is not recommended to be used unless for an obvious reason, like bypassing web filters or restrictions.
