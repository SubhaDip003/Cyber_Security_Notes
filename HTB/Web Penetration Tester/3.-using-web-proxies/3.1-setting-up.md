# 3.1 Setting Up

### Download and Install Burp Suite Pro

Github Link: [https://github.com/xiv3r/Burpsuite-Professional](https://github.com/xiv3r/Burpsuite-Professional)



***

## ✅ **Step-by-Step PowerShell Removal (Run as Administrator) for Windows**

***

### **1. Stop any Burp Suite processes**

```powershell
Get-Process -Name burpsuite, java -ErrorAction SilentlyContinue | Stop-Process -Force
```

***

### **2. Remove typical installation directories**

#### **If Burp was installed via installer (.exe / .msi)**

```powershell
# Program Files (64-bit Burp Installer)
Remove-Item "C:\Program Files\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue

# Program Files (x86)
Remove-Item "C:\Program Files (x86)\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
```

#### **If you downloaded Burp as a standalone JAR**

```powershell
Remove-Item "C:\burp*" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "C:\Users\$env:USERNAME\Desktop\burp*" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "C:\Users\$env:USERNAME\Downloads\burp*" -Recurse -Force -ErrorAction SilentlyContinue
```

***

### **3. Delete user-related Burp files (Extensions, Configs, Logs)**

```powershell
Remove-Item "$env:APPDATA\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$env:LOCALAPPDATA\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$env:USERPROFILE\.BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$env:USERPROFILE\bappstore" -Recurse -Force -ErrorAction SilentlyContinue
```

***

### **4. Delete Java cache directories used by Burp**

```powershell
Remove-Item "$env:LOCALAPPDATA\Temp\burp*" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "$env:LOCALAPPDATA\Temp\java*" -Recurse -Force -ErrorAction SilentlyContinue
```

***

### **5. Delete Burp’s Registry Entries**

⚠ **Registry deletion requires admin rights — copy-paste exactly**.

#### **Uninstall entries**

```powershell
Remove-Item "HKLM:\SOFTWARE\Microsoft\Windows\CurrentVersion\Uninstall\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "HKCU:\Software\Microsoft\Windows\CurrentVersion\Uninstall\BurpSuite" -Recurse -Force -ErrorAction SilentlyContinue
```

#### **Burp-related keys**

```powershell
Remove-Item "HKCU:\Software\JavaSoft\Prefs\burp" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "HKCU:\Software\PortSwigger" -Recurse -Force -ErrorAction SilentlyContinue
Remove-Item "HKLM:\Software\PortSwigger" -Recurse -Force -ErrorAction SilentlyContinue
```

***

### **6. Optional: Remove Java Runtime Environment if installed for Burp**

⚠ Only run if you **want to remove Java too**:

```powershell
Get-WmiObject -Class Win32_Product | Where-Object { $_.Name -like "*Java*" } | ForEach-Object { $_.Uninstall() }
```

***

### **7. Empty Recycle Bin (optional)**

```powershell
Clear-RecycleBin -Force
```

***

### ✅ **One-Click Script (Full Uninstall)**

If you want a single combined script, here it is:

```powershell
# ===== FULL BURP SUITE CLEAN REMOVAL SCRIPT =====

# Optional: clean recycle bin
Clear-RecycleBin -Force

# ===== END SCRIPT =====
```

## ✅ **Step-by-Step Burp Suite Removal (Run as Administrator) for Linux**

***

### ✅ **STEP 1 — Stop any running Burp processes**

```bash
pkill -f burp
pkill -f burpsuite
pkill -f java
```

***

### ✅ **STEP 2 — If installed via apt / Kali repo (Community Edition)**

Check if installed:

```bash
dpkg -l | grep -i burp
```

If listed, remove:

```bash
sudo apt remove --purge burpsuite -y
sudo apt autoremove -y
```

***

### ✅ **STEP 3 — If installed via .sh installer (Professional or Community)**

Installer usually places Burp here:

```bash
sudo rm -rf /opt/BurpSuite
sudo rm -rf /opt/burpsuite
sudo rm -rf /usr/local/BurpSuite
sudo rm -rf /usr/local/burpsuite
```

Some versions install into:

```bash
sudo rm -rf /usr/bin/burpsuite
sudo rm -rf /usr/bin/BurpSuite
```

***

### ✅ **STEP 4 — Remove manually downloaded JAR versions**

```bash
sudo find / -type f -iname "burpsuite*.jar" -delete
sudo find /root -type f -iname "burpsuite*.jar" -delete
sudo find $HOME -type f -iname "burpsuite*.jar" -delete
```

***

### ✅ **STEP 5 — Remove Burp Suite config, extensions, logs (user data)**

Burp stores user configs in:

```bash
rm -rf ~/.BurpSuite
rm -rf ~/.burpsuite
rm -rf ~/.java/.userPrefs/burp
rm -rf ~/.java/.userPrefs/com/portswigger
rm -rf ~/.local/share/BurpSuite
rm -rf ~/.cache/BurpSuite
rm -rf ~/.config/BurpSuite
```

Remove BApp Store extensions:

```bash
rm -rf ~/bapps
rm -rf ~/.local/share/bappstore
```

***

### ✅ **STEP 6 — Remove desktop entry & menu launcher**

```bash
sudo rm -f /usr/share/applications/burpsuite.desktop
sudo rm -f ~/.local/share/applications/burpsuite.desktop
```

***

### ✅ **STEP 7 — Remove Snap version (if installed)**

Check:

```bash
snap list | grep -i burp
```

Remove:

```bash
sudo snap remove burpsuite
```

***

### ✅ **STEP 8 — Remove cached temporary files**

```bash
rm -rf /tmp/burp*
rm -rf /var/tmp/burp*
```

***

### ✅ **STEP 9 — Clean unused dependencies**

```bash
sudo apt autoremove --purge -y
sudo apt clean
```

***

### ⭐ **ONE-CLICK FULL CLEANUP SCRIPT**

If you want everything in one script:

```bash
#!/bin/bash

pkill -f burp 2>/dev/null
pkill -f java 2>/dev/null

sudo apt remove --purge burpsuite -y 2>/dev/null
sudo apt autoremove -y

sudo rm -rf /opt/BurpSuite /opt/burpsuite /usr/local/BurpSuite /usr/local/burpsuite
sudo rm -f /usr/bin/burpsuite /usr/bin/BurpSuite

sudo find / -type f -iname "burpsuite*.jar" -delete 2>/dev/null

rm -rf ~/.BurpSuite ~/.burpsuite ~/.java/.userPrefs/burp
rm -rf ~/.java/.userPrefs/com/portswigger
rm -rf ~/.local/share/BurpSuite ~/.cache/BurpSuite ~/.config/BurpSuite
rm -rf ~/bapps ~/.local/share/bappstore

sudo rm -f /usr/share/applications/burpsuite.desktop
rm -f ~/.local/share/applications/burpsuite.desktop

sudo snap remove burpsuite 2>/dev/null

rm -rf /tmp/burp* /var/tmp/burp*

sudo apt autoremove --purge -y
sudo apt clean
```

Save as:

```bash
nano burp_cleanup.sh
```

Make executable:

```bash
chmod +x burp_cleanup.sh
```

Run:

```bash
./burp_cleanup.sh
```
